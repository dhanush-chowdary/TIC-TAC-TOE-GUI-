# 🎮 Tic Tac Toe – Python GUI

A simple **Tic Tac Toe game with a graphical interface** built using **Python and Tkinter**.
Two players can play the game on the same computer with an interactive 3×3 grid.

---

## 📌 Features

* 🟢 Simple **3x3 Tic Tac Toe board**
* 🎨 Different colors for **Player X** and **Player O**
* 🏆 Automatic **winner detection**
* 🔄 **Reset button** to restart the game
* 🖥️ Built using **Tkinter GUI**

---

## 🛠️ Technologies Used

* **Python**
* **Tkinter (GUI Library)**

## 🎯 How to Play

1. Player **X** starts the game.
2. Players take turns clicking on empty cells.
3. First player to align **3 symbols in a row, column, or diagonal wins**.
4. Click **Reset** to start a new game.
